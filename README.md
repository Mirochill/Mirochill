## Beyond the code
<img src="https://raw.githubusercontent.com/Tarikul-Islam-Anik/Telegram-Animated-Emojis/main/Smileys/Star%20Struck.webp" alt="Star struck" width="27" height="27" align="right">

I do not define myself by a programming stack. I focus on the **idea**, the **experience**, and the **direction** behind a project — then use AI and modern tools to turn ambitious concepts into working prototypes.

What interests me most is the space between imagination and execution: finding the angle that makes a project memorable, shaping a clear interface, testing unusual ideas, and refining the details until everything feels intentional.
